A production-style ecommerce web application where users can browse products, add items to cart, and complete secure online payments, while admins manage products and orders through a protected dashboard. ## Features
- User authentication (Firebase Auth)
- Protected admin routes
- Admin dashboard (product & order management)
- Cart & checkout flow
- Cashfree payment integration
- Order success & failure handling
- Responsive UI

## Tech Stack
- React
- Tailwind CSS
- Firebase (Auth + Firestore)
- Cashfree Payment Gateway
- Vite
